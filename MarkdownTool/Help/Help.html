📖 MarkdownTool Hilfe

Ein nativer macOS Markdown-Editor — schlank, schnell und ohne Ablenkung.

🚀 Schnellstart

1. Ordner öffnen

Verwende ⌘O um einen Ordner mit Markdown-Dateien zu öffnen. Diese erscheinen in der Seitenleiste links.

2. Datei auswählen

Klicke auf eine .md Datei in der Seitenleiste, um sie im Editor zu öffnen. Bei mehreren Dateien werden diese als Tabs angezeigt.

3. Bearbeiten & Vorschau

Schreibe Markdown auf der linken Seite — die formatierte Vorschau erscheint rechts in Echtzeit.

4. Speichern

Drücke ⌘S um die Datei zu speichern. Du kannst auch ⌘⇧S für Export in andere Formate verwenden.

💡 Tipp: Die App speichert automatisch alle 30 Sekunden. Bei einem Absturz werden deine Änderungen wiederhergestellt.

🔧 Werkzeuge

Formatierungs-Toolbar

Die Toolbar unter dem Dateinamen bietet Schnellformatierung:

  • H1, H2, H3 — Überschriften einfügen
  • B, I, S — Fett, Kursiv, Durchgestrichen
  • </> — Code inline
  • { } — Code-Block
  • ☰ — Liste
  • " — Zitat
  • 🔗 — Link einfügen
  • — — Trennlinie

Vorlagen

Drücke ⌘⇧N für Vorlagen wie:

  • README.md
  • Changelog
  • Meeting-Notizen
  • Blog-Post
  • Todo-Liste

Schnellöffnen

Mit ⌘P kannst du schnell durch alle Dateien im geöffneten Ordner navigieren.

Befehlspalette

Mit ⌘⌘ öffnest du die Befehlspalette mit Zugriff auf alle Funktionen:

  • Datei-Befehle (Neu, Öffnen, Speichern, Export)
  • Bearbeiten-Befehle (Rückgängig, Suchen, Ersetzen)
  • Ansicht-Befehle (Vorschau, Seitenleiste, Schriftgröße)
  • Werkzeuge (Theme, Snippets, Statistiken, Backup)

Snippets

Wiederkehrende Texte können als Snippets gespeichert werden. Tippe den Trigger im Editor, z.B.:

  • !readme — README Header einfügen
  • !table — Markdown-Tabelle einfügen
  • !code — Code-Block einfügen
  • !todo — Todo-Liste einfügen

Statistiken

Zeigt Wortanzahl, Zeichen, Absätze und Lesezeit deines Dokuments.

Markdown-Lint

Prüft dein Markdown auf Stil-Probleme und bietet Verbesserungsvorschläge.

Vim-Modus

Aktiviere den Vim-Editor-Modus für schnelles Bearbeiten mit Tastatur:

  • i — Einfügemodus
  • ESC — Normalmodus
  • h j k l — Navigation
  • w b — Wort vor/zurück
  • 0 $ — Zeilenanfang/ende
  • dd — Zeile löschen
  • yy — Zeile kopieren
  • p — Einfügen
  • /text — Suchen
  • n N — Nächste/Vorherige Treffer

🎨 Themes

Wechsle zwischen Dark und Light Mode über die Befehlspalette (⌘⌘) → "Theme wählen".

☁️ Backup & Cloud

Auto-Save sichert deine Änderungen automatisch. In den Backup-Einstellungen kannst du:

  • Auto-Save Intervall anpassen (Standard: 30 Sekunden)
  • Backup-Ordner wählen
  • iCloud-Backup aktivieren

📤 Export

Exportiere dein Markdown in verschiedene Formate:

  • PDF — Für Druck und Teilen
  • HTML — Für Webseiten
  • RTF — Rich Text Format
  • Markdown — Originalformat
